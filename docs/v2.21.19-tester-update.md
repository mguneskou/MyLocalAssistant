# Tester update - v2.21.19

Please move to MyLocalAssistant 2.21.19.0 for this validation cycle.

## What changed in v2.21.19.0

- Fixed work-folder persistence reliability:
  - work-folder paths now support common user forms like `%USERPROFILE%\\...` and `~\\...`
  - paths are normalized to absolute local/UNC paths
  - server now validates that the final path is writable before saving
- Fixed embedding model load reliability on lower-capacity machines:
  - embedding startup now retries with progressively smaller context sizes
  - if GPU probe/load fails, loader retries with CPU-only mode
  - failure messages now point to smaller embedding-model options for low-memory setups
- Improved prompt governance for tool usage:
  - all seeded agents now include explicit Thought/Action/Observation guidance
  - prompts now explicitly require using the matching tool for concrete file deliverables
  - prompts now require explicit tool-name + reason when a required tool is unavailable or fails
