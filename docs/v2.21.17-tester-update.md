# Tester update - v2.21.17

## What changed in v2.21.17.0

- Fixed version drift in runtime surfaces:
  - tray About now reports the correct release version
  - feedback form "Version tested" now matches the release version
  - web Chat UI now shows runtime version in the left sidebar footer
  - web Admin UI now shows runtime version in the left sidebar header
- Fixed release packaging/version stamping drift:
  - release and publish scripts now stamp Server and ServerHost binaries with the resolved release tag version
  - this prevents old embedded version values from leaking into new builds
- Improved Models tab metadata visibility:
  - added Min RAM column
  - added License column (with clickable license link when available)
