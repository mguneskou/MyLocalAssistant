# Tester update - v2.21.7

Please move to MyLocalAssistant 2.21.7.0 for the next test cycle.

## What changed

- Excel workbook automation is broader and more reliable. The app now supports calculation mode control, workbook recalculation, formula evaluation, native refreshable PivotTables, and richer worksheet dashboard elements such as images, hyperlinks, comments, text boxes, and shapes.
- Excel charting is significantly expanded. Native charts now cover stacked column/bar, area, doughnut, scatter, and combo charts, plus legend placement, data labels, axis titles, and per-series colors.
- PowerPoint slide authoring is more robust for branded templates. Placeholder-aware updates now handle reordered template placeholders more reliably, and PowerPoint chart visuals now include line and pie views.
- Office workflow guidance has been tightened so the assistant is more likely to choose the right office tools and follow repeatable template-driven flows.

## Main things to test

- Excel dashboard generation from realistic source data, especially formulas, recalculation, PivotTables, and mixed chart layouts.
- Excel workbook polishing flows that add comments, links, callouts, images, and simple shapes after the data is written.
- PowerPoint generation from an existing branded template, especially slides where title/body placeholders are not in the usual order.
- End-to-end office tasks where the assistant should read from the work directory, produce the workbook or presentation, and save the result back in the expected location.

## How to update

- Installed build: tray icon -> Check for updates...
- Fresh install: use the next 2.21.7.0 installer package when it is published.
- After updating, verify the version from the tray app About window.

## What to send back

Please include:

- the exact version shown in About
- the file or template used for the test
- the prompt you gave the assistant
- whether the output file was created in the expected work location
- screenshots or logs if the result was wrong or incomplete

If you hit a regression, note whether it is an Excel issue, PowerPoint issue, or a work-directory/tool-selection issue.
