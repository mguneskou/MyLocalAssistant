# Tester update - v2.21.13

Please move to MyLocalAssistant 2.21.13.0 for the next validation cycle.

## What changed in v2.21.13.0

- Legacy WinForms client project has been removed from the active product build and solution.
- Product runtime remains web-first: chat at `/` and admin at `/admin`.
- Tray menu wording now uses Open Chat (instead of Open Client) for clearer browser-only flow.
- Admin/chat navigation hardening:
  - admin/global-admin can return to admin from chat via explicit sidebar Open Admin action
  - admin gating now consistently treats admin OR global-admin as admin-capable
- Admin Models tab UX improvements:
  - clear Chat vs Embedding role labels
  - per-row role-specific switching actions (Set Active Chat / Set Active Embedding)
  - explicit active status chips
  - live download progress bar with percent, speed, and ETA
  - auto-refresh while downloads are active
- Embedding catalog expanded with permissive models:
  - Nomic Embed Text v1.5 (Apache-2.0)
  - BGE Base EN v1.5 (MIT)
  - All-MiniLM-L6-v2 (Apache-2.0)
