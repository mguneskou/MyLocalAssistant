# Tester update - v2.21.20

Please move to MyLocalAssistant 2.21.20.0 for this validation cycle.

## What changed in v2.21.20.0

- New CEO Strategic Supervisor seeded agent:
  - added as a restricted built-in agent with id `ceo-strategic-supervisor`
  - designed for Mode A (advisory-only) executive supervision
  - supports recommendation states (`Approve`, `Conditional Approve`, `Reject`), confidence, risk/opportunity framing, evidence mapping, and remediation actions
- New CEO evidence/tool defaults:
  - default tool pack includes `workdir`, `sqlserver`, `word`, `excel`, `powerpoint`, `pdf`, `report.gen`, `time.now`, `math.eval`, and `rag.search_collection`
  - default RAG enabled for the CEO supervisor
- New runtime schema enforcement for CEO Mode A output:
  - system now checks required response sections for the CEO supervisor
  - if sections are missing, the assistant appends a compliance template so output remains action-ready for executive review

## What to test

- Agent visibility and access:
  - in Admin -> Agents, confirm `CEO Strategic Supervisor` exists
  - enable it and assign to a user via the `CEO Strategic Supervisor` department
  - verify non-assigned non-admin users cannot see/use this agent
- Advisory output contract behavior:
  - ask for CEO review of another agent output and confirm response includes:
    - Executive recommendation (Decision + Confidence + rationale)
    - Top risks
    - Top opportunities
    - Evidence map
    - Evidence gaps and assumptions
    - Remediation actions
- Schema enforcement fallback:
  - run prompts likely to produce brief/non-structured responses
  - verify missing-schema cases append the `[Mode A schema compliance notice]` block with required template headings
- Deliverable/tool behavior:
  - request a concrete report output (Word/PDF/Excel)
  - verify tool-backed file generation is attempted and tool failures are named explicitly if they occur
