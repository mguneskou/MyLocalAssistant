# Tester update - v2.21.10

Please move to MyLocalAssistant 2.21.10.0 for the next test cycle.

## What changed

- Legacy WinForms admin app has been removed from the product. Admin operations are now web-only under `/admin`.
- Tray menu `Open Admin` and tray double-click now open the web admin UI, and URL resolution follows configured server listen host/port/scheme.
- Web admin parity is complete for former desktop-admin workflows, including:
  - Usage tab (7/30/90-day stats)
  - Tool plug-in reload
  - Cloud key clear actions
  - Agent prompt test workbench
- Release packaging no longer distributes a desktop admin executable; server + tray host are the admin runtime path.

## Main things to test

- Tray behavior:
  - Right-click tray icon -> `Open Admin` opens browser to `/admin`
  - Double-click tray icon opens `/admin`
- Admin workflow in browser only:
  - users, departments, roles, models, RAG, tools, agents, settings, audit, usage
- Owner-only controls:
  - global system prompt save
  - cloud key set/test/clear
  - tool reload and stats reset
  - agent prompt test modal
- End-to-end validation:
  - change admin settings and verify behavior in chat
  - run audit filters and CSV export

## How to update

- Installed build: tray icon -> Check for updates...
- Fresh install: use the 2.21.10.0 installer package when it is published.
- After updating, verify the version from the tray app About window.

## What to send back

Please include:

- the exact version shown in About
- whether tray `Open Admin` and double-click opened `/admin` correctly
- which admin tabs/workflows you tested
- expected result vs actual result
- screenshots and logs for any failures
