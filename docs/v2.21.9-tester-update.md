# Tester update - v2.21.9

Please move to MyLocalAssistant 2.21.9.0 for the next test cycle.

## What changed

- Web Admin now includes the full operations surface under `/admin`: Users, Departments, Agents, Tools, RAG, and Audit, in addition to Overview, Models, and Settings.
- Owner-level controls are now available in the web Settings page: global system prompt editing, cloud key status, cloud key updates, and provider connection tests.
- Agent management is now available in web admin, including enabled state, model binding, RAG bindings, tool bindings, max tool calls, system prompt, and scenario notes.
- Tool management is now available in web admin, including enable/disable, config JSON updates, tool call stats view, and stats reset.
- RAG administration is now available in web admin: collection create/update/delete, document upload/delete, and grant management for User/Department/Role principals.
- Audit operations are now available in web admin with filterable paging and CSV export.

## Main things to test

- Full admin workflow in browser only: create/update users, reset passwords, and verify self-protection rules (cannot disable/delete current account).
- Owner-only controls: update global system prompt, save cloud keys, and run provider tests.
- Agent updates in web admin: model/tool/RAG bindings, system prompt and scenario notes, then verify behavior in chat.
- Tool operations: edit config JSON, enable/disable selected tools, and verify stats/reset behavior.
- RAG flow: create collection, upload docs, add/remove grants, and validate retrieval behavior in chat.
- Audit flow: apply filters (time/action/user/success/admin scope), paginate results, and export CSV.

## Known scope note

- Plug-in reload is not exposed by the current server API. This release includes tool stats reset in web admin, but not plug-in reload.

## How to update

- Installed build: tray icon -> Check for updates...
- Fresh install: use the 2.21.9.0 installer package when it is published.
- After updating, verify the version from the tray app About window.
